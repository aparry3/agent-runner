# Evals Dashboard

The Evals Dashboard in the Studio provides a visual interface for running and reviewing agent evaluations.

## Features

- **Run Evals** — Execute eval suites with a single click
- **Results Table** — Pass/fail status for each test case with scores
- **Assertion Details** — Expand any test case to see individual assertion results
- **History** — Track results over time to catch regressions

## Running Evals

1. Select an agent from the list
2. Click **"Run Evals"**
3. View results with pass/fail indicators and scores

## Adding Test Cases

You can add test cases directly in the Studio:

1. Open the agent's eval configuration
2. Click **"Add Test Case"**
3. Enter input, expected output, and assertions
4. Save and run

Test cases added in the Studio are saved to the same agent definition that code uses.
